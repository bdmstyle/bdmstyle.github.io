Não é tão fácil usar as últimas versões do lombok, netbeans e maven juntos, eu achei uma boa solucão neste blog post onde tem mais detalhes. Mas eu vou dar uma explicação simplificada aqui e disponibilizar um projeto de exemplo no github.

1- Configure seu pom. {% gist bdmstyle/5c1f13825314ca794688c517706b5526 %}

2 - Em Project Properties->Actions->Activate Profiles coloque "lombok-build" em todas as operações que você use como por exemplo "build".

3- Funcionando! Note que na linha de comando o mvn clean package funciona normalmente (para quem não usa o netbeans como o jenkins ou eclipse).
